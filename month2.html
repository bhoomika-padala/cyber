🖥️ VirtualBox Installation

VirtualBox is virtualization software used to create and run virtual machines.

Why It Matters:

Used to create cybersecurity labs safely without affecting your main system.

Installation Steps:

  1. Download the platform binary from virtualbox.org
  2. Run the executable with Administrator privileges
  3. Enable Virtualization Support (VT-x/AMD-V) in your hardware BIOS
  4. Complete installation setup

🐱 Kali Linux Installation

Kali Linux is a penetration testing Linux distribution.

Why It Matters:

Used for cybersecurity testing and ethical hacking. Comes pre-installed with hundreds of security tools.

Installation Steps:

  1. Download the pre-built 64-bit VirtualBox (.vbox) image
  2. Open VirtualBox → File → Import Appliance
  3. Allocate a minimum of 2048MB RAM and 2 Processor Cores
  4. Select "Copy machine ID"
  5. Click Import

🎯 Metasploitable Installation

Metasploitable is an intentionally vulnerable virtual machine.

Why It Matters:

Used for safe cybersecurity practice. It contains known vulnerabilities that you can practice exploiting in a controlled environment.

Installation Steps:

  1. Extract the downloaded zip file
  2. Open VirtualBox → Click "New"
  3. Name it 'Metasploitable' → Select Linux/Ubuntu 64-bit
  4. Choose "Use an existing virtual hard disk file"
  5. Browse and select the extracted .vmdk file
  6. Click Create

🌐 NAT

VM accesses internet through the host machine. The VM is hidden behind the host's IP address.

🔗 NAT Network

Allows VM communication inside a NAT environment. Multiple VMs can communicate with each other while sharing the host's IP.

🌍 Bridged Adapter

VM appears as a real device on the network with its own IP address. It can communicate directly with other devices on the network.

🔒 Internal Network

Only internal VM communication is allowed. VMs can talk to each other but have no access to the external network or internet.

Host Discovery & Scanning

📡 Netdiscover

Discovers live hosts using ARP (Address Resolution Protocol). Useful for identifying devices on your local network.

⚡ Angry IP Scanner

Fast network scanning tool that can scan IP addresses and ports. Cross-platform and lightweight.

🔍 Advanced IP Scanner

Windows-based network scanner. Provides quick network analysis and remote control capabilities.

📶 Ping Sweep

Technique for identifying live hosts by sending ICMP echo requests to multiple addresses.

Nmap

Nmap (Network Mapper) is a powerful network scanning and enumeration tool used by security professionals worldwide.

Main Uses:

  • Host discovery
  • Port scanning
  • Service detection
  • OS detection
  • Enumeration

Important Nmap Commands:

nmap # Basic scan nmap -F # Fast scan (top 100 ports) nmap -p- # Scan all 65535 ports nmap -sS # Stealth SYN scan nmap -sU # UDP scan nmap -O # OS detection nmap -sV # Service version detection nmap -A # Aggressive scan (OS, version, script, traceroute) nmap -sC # Default script scan

Zenmap

Zenmap is the official graphical user interface (GUI) version of Nmap.

Features:

  • Topology Visualization - Visual representation of network layout and device connections
  • Scan Profiles - Pre-configured scanning templates for different scenarios
  • Result Visualization - Interactive display of scan results with filtering and sorting
