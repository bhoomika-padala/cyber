Logical Ports Reference

Logical ports are communication endpoints used by services and applications. Ports help systems identify which service should receive network traffic.

Total logical ports: 65,535

Standard Port Ranges

Port Range Type
0 – 1023 Well-known Ports
1024 – 49151 Registered Ports
49152 – 65535 Dynamic/Private Ports

Nmap Command Reference

Critical scanning flags for rapid lookups. Nmap is the most popular network scanning tool in cybersecurity.

Command Description
nmap Basic scan of target
nmap -F Fast scan (top 100 ports)
nmap -p- Scan all 65535 ports
nmap -sS Stealth SYN scan
nmap -sU UDP scan
nmap -O OS detection
nmap -sV Service version detection
nmap -A Aggressive scan (OS, version, script, traceroute)
nmap -sC Default script scan

Netcat Cheatsheet

Netcat is the "Swiss Army knife of networking" - a powerful tool for reading from and writing to network connections.

Common Uses:

  • Banner grabbing
  • Manual interaction with services
  • Shell interaction (reverse/bind shells)
Command Description
nc Connect to a host on specified port
nc -l Listen on specified port
nc -lvnp Listen with verbose output, numeric IPs, and no DNS resolution
nc -zv Zero-I/O mode (scanning)
nc -w Set timeout for connections

Software Glossary

An alphabetized glossary of all tools mentioned across the course modules.

Advanced IP Scanner
Windows-based network scanner that provides quick network analysis and remote control capabilities.
Angry IP Scanner
Fast, cross-platform network scanning tool that can scan IP addresses and ports.
Kali Linux
A Debian-based Linux distribution designed for digital forensics and penetration testing. Comes pre-installed with hundreds of security tools.
Metasploitable
An intentionally vulnerable virtual machine created for security training and testing purposes.
Nmap
Network Mapper - A free and open-source utility for network discovery and security auditing. Used for host discovery, port scanning, service detection, and OS detection.
Netcat
A versatile networking tool often called the "Swiss Army knife of networking." Used for reading from and writing to network connections using TCP or UDP.
Netdiscover
An active/passive ARP reconnaissance tool used for discovering live hosts on a local network.
Ophcrack
A Windows password cracking tool that uses rainbow tables for efficient LM/NTLM password recovery.
VirtualBox
A free and open-source virtualization software used to create and run virtual machines on your computer.
Zenmap
The official graphical user interface (GUI) for Nmap. Provides topology visualization, scan profiles, and interactive result display.
